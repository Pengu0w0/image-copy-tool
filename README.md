# image-copy-tool

私が試作で作ったパソコン専用の画像コピーツールです。

## 概要
- 指定したフォルダに、ダウンロードフォルダやピクチャフォルダの画像をまとめてコピーできます。
- コピーするだけなので、元の画像は元の場所に残ります。

## 使い方
1. Python 3 をインストール
2. このリポジトリをクローンまたはダウンロード
3. ターミナルで `python main.py` を実行
4. GUIでコピー先のフォルダを選択

## 注意点
- コピー先フォルダに書き込み権限が必要
- 元のコピーした画像は消えません
